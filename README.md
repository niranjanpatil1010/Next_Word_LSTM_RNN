# Next Word Prediction using LSTM

A deep learning project that predicts the next word in a sentence using LSTM (Long Short-Term Memory) networks.

## Features
- Text preprocessing
- Tokenization
- LSTM-based prediction
- Streamlit/Flask interface

## Tech Stack
- Python
- TensorFlow/Keras
- NumPy
- Pandas

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
